Electronics-Site

Developed a full‑stack electronics e‑commerce application featuring a responsive React.js frontend integrated with Redux for state management, Axios for API calls, Bootstrap & custom CSS for layout and styling, and session persistence via react‑cookies. The backend is built with Node.js and Express, exposing RESTful API endpoints connected to MongoDB Atlas using native drivers. Deployed frontend on GitHub Pages and configured the backend for secure cloud deployment. Implemented form validation, error‑handling, and secure session management to ensure a robust, user‑friendly experience.
